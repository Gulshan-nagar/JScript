Documentation

Welcome to the documentation of our custom programming language. Below is an explanation of the syntax and features:

1. Variables

To declare a variable, use the keyword let followed by the variable name and value:

// Example of variable declaration
let x = "Hello, World!";

2. Loops

Instead of for or while, use the loop keyword:

// Example of a loop
loop (i = 0; i < 10; i++) {
    print(i);
}

3. Functions

To define a function, use the function keyword:

// Example of a function
function greet() {
    print("Hello from the custom IDE!");
}

// Calling the function
greet();

4. Printing Output

Use the print keyword to display output instead of console.log:

// Printing a message
print("This is custom syntax!");

5. Conditional Statements

Use if and else for conditional logic:

// Example of an if-else statement
if (x == "Hello") {
    print("Condition is true");
} else {
    print("Condition is false");
}
